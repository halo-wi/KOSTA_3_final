# SNACK_24 - - Kosta213(3조)
---------
## 개발자
* 김성휘
* 고석우
* 강성빈
* 남후승
* 임세혁

## 프로젝트 계획이유
> 많은 구독 서비스들이 눈앞에 쏟아지는 가운데 트렌드에 따라서 간식 구독 서비스를 만들어보면 좋을 것 같아 프로젝트를 기획/개발하게 되었습니다. 실제 배송은 안되지만 테스트 결제 시 구독 및 정기 결제까지 구현하였습니다.
  
## 개발환경
* 사용언어:Java1.8, javaScript(ECMA6)
* DB: Oracle 11
* 자료공유:GitHub
* front-End : thymeleaf, JQuery, Ajax, Bootstrap
* back-End : Spring Boot, Spring security
* 배포 : AWS (Ec2, Docker)
